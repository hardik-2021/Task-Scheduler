# Task Scheduler
Task Scheduler is a responsive React.js task management app that lets users create, view, and manage daily tasks with ease. Built using a component-driven approach, it focuses on core React concepts like state management, props, component reuse, and clean UI structuring.

# Steps to run:
1. Download the zip file.
2. Extract it.
3. Open it in VS code or any code editor or terminal.
4. Install node_modules using "npm install" command in the terminal.
5. After installation, run the web-application on your local server by using command "npm start".
6. The application will start running in browser.
7. Use the task scheduler.
